Tabs

Tabs

When you click on tab, only the container with the corresponding tab id will become visible.

You can add the class .disabled to make a tab inactivated.

If you want to use a precise external link to indicate that the tab will not have section [stewed-tab-section="false"].

To use tabs you nead use the class .tabs, .tabs__item and .tab-section.

But also need to use attributes to be able to identify your tabs [stewed-tabs="ID_CONTAINER"] and [stewed-tab-id="ID_TAB"].
