SideNav

Use SideNav simply create a container with .js-side-nav and .side-nav then you need to create an action with the class .js-menu-show that will show your sidenav, which is hidden.


To close the SideNav user just click an area outside the sidenav or if you prefer you can create an action to close by simply adding the .js-menu-hide class
