Button Default

Button Colors

Optional, can use the buttons components with this specific colors applying the next modifiers.

  • .btn--default
  • .btn--danger
  • .btn--success
  • .btn--info

Disabled Icon

Optional, for the icon element

  • btn--icon

Optional, can add disabled directly to the button element to disable a button

  • disabled
