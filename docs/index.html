Use Stewed

Getting started

                    <!-- Compiled and minified CSS -->
                    <link rel="stylesheet" type="text/css" media="screen" href="bower_components/dist/css/stewed.min.css" />

                    <!-- Compiled and minified JavaScript -->
                    <script src="bower_components/dist/js/stewed.min.js">
