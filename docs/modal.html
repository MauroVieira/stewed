Introduction

Modal is positioned overlapping within the display window and is protected page content.

Show Modal Effect

Modal HTML Structure

Learn to cook
                            <!-- Use a hyperlink 'href=""' -->
                            <a class="button button-white button button__large openModal" href="#modal">Show Modal</a>

                            <!-- Use a button 'stewed-modal=""' -->
                            <button class="button button-white button button__large openModal" stewed-modal="modal">Show Modal</button>

                            <!-- Structure -->
                            <div id="modal" class="container modal">
                            <div class="modal-head">
                            <h2>Modal Header</h2>
                            </div>
                            <div class="modal-body">
                            <p>...</p>
                            </div>
                            <div class="modal-footer">
                            <!-- Use class 'modal-close' to close modal on click -->
                            <a href="#!" class="modal-close button button-white button button__large float-right">Disagree</a>
                            <a href="#!" class="button button-white button button__large float-right">Agree</a>
                            </div>
                            </div>
