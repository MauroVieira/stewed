Tooltip

Last Updated: 26 Nov 2017

Tooltip Default

To use the tooltip simply just use the [data-tooltip] in your tag, but if you want to change the direction of it, should one of the following classes:

  • .tooltip--top
  • .tooltip--right
  • .tooltip--bottom
  • .tooltip--left
