# Menu

Menu display a list of choices that can be selected.

### Installation

```bash
npm install --save @stewed/components
```

### Import

```js
import { Menu } from '@stewed/components/menu';
```

## Examples

export const Template = (args) => (
    <Menu {...args}>
        <Menu.Item>Menu 1</Menu.Item>
        <Menu.Item selected>Menu 2</Menu.Item>

        <Menu.Item>Menu 4</Menu.Item>
        <Menu.Separator>Separator</Menu.Separator>
