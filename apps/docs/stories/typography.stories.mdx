# Typography

Typography is the used to render headings and paragraphs within an interface.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Typography } from '@stewed/react';
```

## Examples

export const Template = (args) => <Typography {...args} />;

<Canvas>
