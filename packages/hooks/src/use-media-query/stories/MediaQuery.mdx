# useMediaQuery

Hook that listens to a CSS media query and returns a boolean indicating whether the query currently matches the viewport.

### Installation

```bash
npm install --save @stewed/hooks
```

### Import

```js
import { useMediaQuery } from "@stewed/hooks";
```
