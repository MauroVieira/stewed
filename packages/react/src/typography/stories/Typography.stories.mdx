# Text

Text is the used to render headings and paragraphs within an interface.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Text } from "@stewed/react";
```

## Examples

export const Template = (args) => (
  <Theme>
    <Text {...args} />
  </Theme>
);
