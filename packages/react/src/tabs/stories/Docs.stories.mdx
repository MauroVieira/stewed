# Tabs

Navigation between multiple pages or content sections.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Tabs } from "@stewed/react";
```

### Usage

```tsx
<Tabs value="1">
  <Tabs.List>
    <Tabs.Item value="1">Item</Tabs.Item>
    <Tabs.Item value="2">Item Long</Tabs.Item>
    <Tabs.Item value="3">Item Very Long</Tabs.Item>
  </Tabs.List>
</Tabs>
```
