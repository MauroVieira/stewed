# Hue

A React component that manages the backgrounds for its children. 

It allows you to define and control the background transitions, providing flexible styling for backgrounds within your components.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Hue } from "@stewed/react";
```
