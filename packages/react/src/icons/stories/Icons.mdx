# Icon

The Icon component is used internally by other components, such as `<Checkbox />` , `<Dialog />` or `<Pagination />` to render scalable vector icons. 

It can also be used externally, offering flexibility to adapt to different sizes, colors, and use cases.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Icon } from "@stewed/react";
```
