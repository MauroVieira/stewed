# Alert

An alert displays a short and important message in a way that attracts the user attention.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Alert } from '@stewed/react';
```

## Examples

export const Template = (args) => <Alert {...args} />;
