# Skeleton

Skeleton is a animated placeholder, should be used to render content placeholders before the real data is available.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Skeleton } from "@stewed/react";
```

## Examples

export const Template = (args) => (
  <Theme>
    <Skeleton {...args} />
  </Theme>
);
