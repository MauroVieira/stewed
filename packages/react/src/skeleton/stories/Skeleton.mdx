# Skeleton

Skeleton is a animated placeholder component used to render content placeholders before the actual data or content is available.

It helps improve the user experience by indicating that content is loading.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Skeleton } from "@stewed/react";
```
