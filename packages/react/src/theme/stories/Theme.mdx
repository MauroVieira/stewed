# Theme

A theme component allows you to manage various objects that define your application's colors, spacing, fonts, and more in a coherent and organized manner. 

By centralizing these design elements within a theme component, you can ensure consistency and ease of maintenance throughout your application. 

This approach enables you to easily update and modify the visual aspects of your application by making changes within the theme component, 
leading to a more streamlined and efficient design process.

### Installation

```bash
npm install --save @stewed/react
```

### Import

```js
import { Theme } from "@stewed/react";
```

### Usage

```tsx
<Theme<"default" | "epic">
  tokens={{
    default: {
      color: {
        "primary": "#3f51b5",
      },
      fontFamily: {
        base: "'Roboto Serif', serif",
      },
    },
    epic: {
      fontFamily: {
        base: "'DM Sans', sans-serif",
      },
      color: {
        "backdrop": "rgb(200 200 200 / 70%)",
        "primary": "#9c27b0",
        "secondary": "#000",
        "critical": "#e91111",
        "success": "#14784a",
      },
      components: {
        "text-field": {
          radius: "none",
        },
        "select": {
          radius: "none",
        },
        "card": {
          radius: "none",
        },
        "switch": {
          radius: "none",
        },
        "dialog": {
          radius: "none",
          shadow: "xl",
        },
        "tabs": {
          radius: "none",
        },
        "button": {
          radius: "none",
        },
      },
    },
  }}
  defaultTheme="epic"
  scoped>
    Children
</Theme>
```
