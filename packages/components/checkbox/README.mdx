# Checkbox

Checkbox allow the user to select one or more items from a set.

```bash
$ npm install --save @stewed/checkbox
```

## Import

### Styles

```scss
@use '~@stewed/checkbox/scss';
```

### Component

```js
import { Checkbox } from '@stewed/checkbox';
```

## Props

checkbox component extends from `React.InputHTMLAttributes<HTMLInputElement>` interface, that provide special properties and methods for manipulating the options.

| Name          | Type                                                | Default   | Description                                      |
| ------------- | --------------------------------------------------- | --------- | ------------------------------------------------ |
| `appearance?` | `One of<"default", "success", "danger", "warning">` | 'default' | The base styling to apply to the component.      |
| `size?`       | `One of<"sm", "md", "lg"`                           | 'md'      | The base size to apply to the component.         |
| `disabled?`   | `boolean`                                           | false     | Set if the component is disabled.                |
| `className?`  | `string`                                            | -         | Add additional class name to the root component. |

The `ref` is forwarded to the root element.

## CSS

| Class name        | Description                                                   |
| ----------------- | ------------------------------------------------------------- |
| `.checkbox`          | Styles applied to the root element.                           |
| `.checkbox--success` | Styles applied to the root element if `appearance="success"`. |
| `.checkbox--danger`  | Styles applied to the root element if `appearance="danger"`.  |
| `.checkbox--warning` | Styles applied to the root element if `appearance="warning"`. |
| `.checkbox--sm`      | Styles applied to the root element if `size="sm"`.            |
| `.checkbox--lg`      | Styles applied to the root element if `size="lg"`.            |
