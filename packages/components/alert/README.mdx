# Alert

An alert displays a short and important message in a way that attracts the user attention.

```bash
$ npm install --save @stewed/alert
```

## Import

### Styles

```scss
@use '~@stewed/alert/scss';
```

### Component

```js
import { Alert } from '@stewed/alert';
```

## Props

| Name          | Type                                             | Default | Description                                      |
| ------------- | ------------------------------------------------ | ------- | ------------------------------------------------ |
| `appearance?` | `One of<"info", "success", "danger", "warning">` | 'info'  | The base styling to apply to the component.      |
| `className?`  | `string`                                         | -       | Add additional class name to the root component. |
| `title?`      | `string`                                         | -       | Slot for title content.                          |
| `children?`   | `React.ReactNode`                                | -       | The value displayed within the component.        |

## CSS

| Class name        | Description                                                   |
| ----------------- | ------------------------------------------------------------- |
| `.alert`          | Styles applied to the root element.                           |
| `.alert__title`   | Styles applied to the title wrapper element.                  |
| `.alert__body`    | Styles applied to the body wrapper element.                   |
| `.alert--info`    | Styles applied to the root element if `appearance="info"`.    |
| `.alert--success` | Styles applied to the root element if `appearance="success"`. |
| `.alert--danger`  | Styles applied to the root element if `appearance="danger"`.  |
| `.alert--warning` | Styles applied to the root element if `appearance="warning"`. |
