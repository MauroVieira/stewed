# Badge

A badge consists of a small circle notification element.

```bash
npm install --save @stewed/badge
```

## Import

### Styles

```scss
@use "~@stewed/badge/scss";
```

### Component

```js
import { Badge } from "@stewed/badge";
```

## Props

| Name          | Type                                                                      | Default   | Description                                      |
| ------------- | ------------------------------------------------------------------------- | --------- | ------------------------------------------------ |
| `appearance?` | `"primary" \| "secondary" \| info" \| "success" \| "danger" \| "warning"` | "primary" | The base styling to apply to the component.      |
| `className?`  | `string`                                                                  | -         | Add additional class name to the root component. |
| `count?`      | `string`                                                                  | -         | The content rendered within the badge.           |
| `children?`   | `React.ReactNode`                                                         | -         | The badge will be added relative to this node.   |

## CSS

| Class name          | Description                                                     |
| ------------------- | --------------------------------------------------------------- |
| `.badge`            | Styles applied to the root element.                             |
| `.badge--primary`   | Styles applied to the root element if `appearance="primary"`.   |
| `.badge--secondary` | Styles applied to the root element if `appearance="secondary"`. |
| `.badge--info`      | Styles applied to the root element if `appearance="info"`.      |
| `.badge--success`   | Styles applied to the root element if `appearance="success"`.   |
| `.badge--danger`    | Styles applied to the root element if `appearance="danger"`.    |
| `.badge--warning`   | Styles applied to the root element if `appearance="warning"`.   |
