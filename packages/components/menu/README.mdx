# Menu

Menu display a list of choices that can be selected.

```bash
$ npm install --save @stewed/menu
```

## Import

### Styles

```scss
@use '~@stewed/menu/scss';
```

### Component

```js
import { Menu } from '@stewed/menu';
```

## Props

### Menu

| Name         | Type              | Default | Description                                      |
| ------------ | ----------------- | ------- | ------------------------------------------------ |
| `className?` | `string`          | -       | Add additional class name to the root component. |
| `children?`  | `React.ReactNode` | -       | The value displayed within the component.        |

### Menu Item

| Name          | Type                          | Default   | Description                                      |
| ------------- | ----------------------------- | --------- | ------------------------------------------------ |
| `className?`  | `string`                      | -         | Add additional class name to the root component. |
| `appearance?` | `One of<"default", "danger">` | 'default' | The base styling to apply to the component.      |
| `selected?`   | `boolean`                     | false     | Set if the component is selected.                |
| `disabled?`   | `boolean`                     | false     | Set if the component is disabled.                |
| `children?`   | `React.ReactNode`             | -         | The value displayed within the component.        |

### Menu Separator

| Name         | Type     | Default | Description                                      |
| ------------ | -------- | ------- | ------------------------------------------------ |
| `className?` | `string` | -       | Add additional class name to the root component. |

## CSS

| Class name              | Description                                                          |
| ----------------------- | -------------------------------------------------------------------- |
| `.menu`                 | Styles applied to the root element.                                  |
| `.menu__item`           | Styles applied for the list item element.                            |
| `.menu__item--danger`   | Styles applied to the **MenuItem** element if `appearance="danger"`. |
| `.menu__item--selected` | Styles applied to the **MenuItem** element if `selected={true}`.     |
| `.menu__item--disabled` | Styles applied to the **MenuItem** element if `disabled={true}`.     |
