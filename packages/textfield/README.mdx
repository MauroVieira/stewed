# Textfield

A textfield is an input that allows a user to write or edit text.

```bash
$ npm install --save @stewed/textfield
```

## Style

```scss
@use '~@stewed/textfield/scss';
```

## Props

Textfield component extends from `React.InputHTMLAttributes<HTMLInputElement>` interface, that provide special properties and methods for manipulating the options.

| Name          | Type                                                | Default   | Description                                      |
| ------------- | --------------------------------------------------- | --------- | ------------------------------------------------ |
| `appearance?` | `One of<"default", "success", "danger", "warning">` | 'default' | The base styling to apply to the component.      |
| `className?`  | `string`                                            | -         | Add additional class name to the root component. |

The `ref` is forwarded to the root element.

## CSS

| Class name           | Description                                                     |
| -------------------- | --------------------------------------------------------------- |
| `.textfield`            | Styles applied to the root element.                             |
| `.textfield--success`   | Styles applied to the root element if `appearance="success"`.   |
| `.textfield--danger`    | Styles applied to the root element if `appearance="danger"`.    |
| `.textfield--warning`   | Styles applied to the root element if `appearance="warning"`.   |
