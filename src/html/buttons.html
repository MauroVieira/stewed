Button Default

Mandatory, defaults to a text button

  • .btn

Button Colors

Optional, can use the buttons components with this specific colors applying the next modifiers.

  • .btn--default
  • .btn--danger
  • .btn--success
  • .btn--info

Button Icon

Optional, for the icon element

  • btn--icon

Disabled Button

Optional, can add disabled directly to the button element to disable a button

  • disabled

Button Sizes

Optional, can use the buttons components with this specific size applying the next modifiers.

  • .btn--small
  • .btn--medium
  • .btn--large
