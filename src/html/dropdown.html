Dropdown

To use the dropdown create a div with the class .dropdown then only needs to use a button with the attribute [stewed-dropdown] and class .dropdown__button.


You can set the alignment of the dropdown with .dropdown--top-left and .dropdown--top-right in your div dropdown.

Then simply create a container that contains the id attribute clearcut the date dropdown button.


The button to be used may be any, provided that uses the attribute.


You can customize the colors of the buttons, as well as the dropdwons.
